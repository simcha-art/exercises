# Factory Functions & Closures – תרגילים

---

## חלק א' – בסיס: Factory Functions

---

### תרגיל 1

צור פונקציה שמקבלת שם ומחזירה אובייקט עם השדה `name`.

פלט צפוי: אובייקט עם `name`.

---

### תרגיל 2

הוסף לאובייקט פונקציה `greet` שמחזירה `"היי, אני [name]"`.

</div>

```
פלט: "היי, אני דנה"
```

---

<div dir="rtl">

### תרגיל 3

צור שני אובייקטים שונים עם שמות שונים ובדוק שהם לא משפיעים אחד על השני.

פלט צפוי: שינוי באחד לא משנה את השני.

---

### תרגיל 4

הוסף שדה `age` לאובייקט והחזר אותו.

</div>

```
פלט: { name: "דנה", age: 28 }
```

---

<div dir="rtl">

### תרגיל 5

הוסף פונקציה שמחזירה האם המשתמש מעל גיל 18.

</div>

```
פלט: true או false
```

---
